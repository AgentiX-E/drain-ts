@agentix-e/drain-ts

A TypeScript/Node.js streaming log template miner based on the Drain algorithm — ported with high fidelity from the official Python Drain3 v0.9.11.

GitHub Repository npm Package Coverage Report

About

drain-ts extracts structured templates from unstructured log messages in real time. It employs a fixed-depth prefix tree with similarity-based clustering to identify constant and variable parts of log lines — the same algorithm powering the widely-cited Drain3 Python implementation.

Features

  • Full Drain3 parity — treeSearch, fastMatch, LRU eviction, match inference, parameter extraction
  • Streaming online log processing with O((d + cm)n) complexity
  • Zero runtime dependencies — no mandatory peer deps
  • Cross-runtime: Node.js ≥22, Deno, Bun, browsers
  • Framework-agnostic state persistence (File, Redis, Kafka, S3)
  • Built-in wall-clock profiler

Quick Start

import { TemplateMiner, TemplateMinerConfig, DEFAULT_MASKING_INSTRUCTIONS } from "@agentix-e/drain-ts";

const miner = new TemplateMiner({
  config: TemplateMinerConfig.from({ maskingInstructions: DEFAULT_MASKING_INSTRUCTIONS }),
});

miner.addLogMessage("connection from 192.168.1.1 port 8080");
// Template: "connection from <IP> port <NUM>"
